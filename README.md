## Webcraft description

This is a web copy of Starcraft 2. You can build units and structures there. You
can control units (zerglings, drones and acilings). You can build spawning pool, evolution chamber, slime generator or mining tower to boost your economy.
You can change the type of the slime generators' slime to increase the speed or HP.
You can upgrade and evolve your units and structures, too. You can fight with enemies (zerglings, zealots and acilings).
There are lots of waves of zerg enemies. There are sounds. There are lots of obstacles. There are multiple selecting mechanisms. The playing field of this game is automaticly set. A song will soon be added, too.

You can pause the game or move the screen.

Webcraft is in progress.
