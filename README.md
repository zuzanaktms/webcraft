## Webcraft description

This is a web copy of starcraft 2. There you can build units and structures you
can control units (zerglings,drones and acilings). You can build spawning pool,evolution chamber or sime generator.
You can change type of slime generator slime to increase speed or HP.
You can upgrade and evolve your units and structures too. You can fight with enemy(zerglings,zealots and acilings).
There are lots of waves of enemis zregs or protosses. There are sounds. This game is automaticly set size of 
playing field.

You can pause the game or move the screen.

Webcraft is in progress.
