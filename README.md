## Webcraft description

This is a web copy of starcraft 2. There you can build units and structures you
can control units (zerglings and drones). You can build spawning pool,evolution chamber or defend tower
there you can upgrade and evolve your units and structures. You can fight with enemy(zerglings or zealots).
There is one wave of enemis zregs or protosses. There are sounds. This game is atomaticly set size of 
playing field.

Webcraft is in progress.
